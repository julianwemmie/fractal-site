# Julian's Personal Site

A personal site built with Claude Code, exploring how AI can be used as a development harness.

**Live:** https://fractal-site.netlify.app

## Features

- Random design on each page load (7 distinct design styles)
- Markdown-based blog posts
- Built with Vite, React, TypeScript, and Tailwind CSS

## Quick Start

```bash
npm install
npm run dev
npm run build
```
